## Troubleshooting
| Problem                        | Solution                                              |
|--------------------------------|-------------------------------------------------------|
| Mode doesn't activate          | Run as Administrator and restart PC                   |
| Controller not detected        | Check Xbox Series X/S controller connection           |
| Performance issues             | Apply performance optimization fixes                  |
| Vivetool command fails         | Run Vivetool setup guide again                        |
| Game Pass not working          | Reinstall Game Pass app and check dev mode            |
| Black screen / crash           | Disable Xbox Mode and restore system point            |
| Handheld console features missing | Update Windows 11 and re-apply interface tweaks     |
